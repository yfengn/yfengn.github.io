react-native run-android 报错Failed to install the following Android SDK

发表于2022-03-26更新于2022-01-11

react-native run-android启动之后发现出现以下错误，导致不能正常启动 Android 工程。

这个问题的主要原因是电脑上之前已经安装过SDK,没有接受 许可协议导致的。

首先进入到sdk路径(…/sdk/tools/bin)，找到可以处理licences的文件，然后打开命令行输入 sdkmanager –licenses

之后会有一些询问(y/N)，请输入y，否则可能无法使用 react-native run-android 启动 android 工程，操作结束后即可启动项目
